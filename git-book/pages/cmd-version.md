# Lampmanのコマンドバージョンを表示する：`lamp version`
----------------------------------------------------------------------

### `lamp version`

バージョンと実行モードが表示されます。


<pre class="cmd">
$ lamp version

  ╭───────────────────╮
  │ Lampman ver 1.0.0 │
  ├╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶┤
  │ mode: default     │
  ╘═══════════════════╛
</pre>

### `lamp version -q`<br>`lamp version --quiet`

バージョンのみのシンプルな１行テキストが表示されます。

<pre class="cmd">
$ lamp version -q
1.0.0
</pre>
