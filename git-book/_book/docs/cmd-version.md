
# lamp version

## Lampmanのコマンドバージョンを表示する

### `lamp version`

バージョンと実行モードが表示されます。


<pre style="font-size:12px;white-space:pre;line-height:1.2em">
$ lamp version

  ╭───────────────────╮
  │ Lampman ver 1.0.0 │
  ├╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶╶┤
  │ mode: default     │
  ╘═══════════════════╛
</pre>

### `lamp version -q`<br>`lamp version --quiet`

バージョンのみ1行のシンプルな文字列で表示されます。

<pre style="font-size:12px;white-space:pre;line-height:1.2em">
$ lamp version -q
1.0.0
</pre>
