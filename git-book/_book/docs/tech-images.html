技術情報：Dockerイメージについて

kazaoki/lampman イメージの仕様

Services/Apps External ports Version 備考
Linux CentOS 7.6
Apache 80, 443 デフォルトでは public_html/ が公開ディレクトリです。
PHP PHP 5.4.16 OSに組み込まれているPHPです。このバージョンは古いので使用はおすすめしません。
Perl/CGI perl 5.16.3
MailDev 9981 MailDev 1.1.0 sendmailに飛んできたメールをキャッチしてブラウザから中身を確認できるツールです。
Postfix Postfix 2.10 メールの送信処理に必要です。（MailDevに飛ばすのにも使用します）

※１サービス１イメージが推奨のDockerですが、LAMP構成を１つの大きなサービスとして１つのイメージにまとめています。

その他に使用されるDockerイメージ

Images Description
kazaoki/phpenv バージョンごとのイメージを DockerHub に用意していますので、 config.js ではここにあるバージョンを指定してください。（本体のlampmanコンテナには phpenv が入っており、PHPバージョンごとに差分のみのイメージになります）
mysql MySQL の公式イメージです。
postgres PostgreSQL の公式イメージです。

※ちなみに、イメージ名は config.js で変更できるので、自作カスタムのイメージに差し替え可能です。
